Netflix Clone (MERN Stack)

A full-stack Netflix-inspired web application built using the MERN stack (MongoDB, Express, React, Node.js).

This project was built while following a tutorial and customizing the implementation for learning purposes.

Features
	•	User authentication (Register / Login)
	•	Browse movies and series
	•	Featured banner with random content
	•	Genre filtering
	•	Video watch page
	•	Admin dashboard for managing content
	•	REST API with JWT authentication

  ech Stack
	•	React
	•	Node.js
	•	Express
	•	MongoDB
	•	SCSS
	•	Material UI

Project Structure

/client  → Frontend  
/api     → Backend  
/admin   → Admin dashboard
